Java List与数组之间的转换

2016-07-20 (Modified: 2020-07-20) — 1 min read
#Java  #List 

1. 数组转换为List

String[] arr = new String[] {"str1", "str2"};
List<String> list = Arrays.asList(arr);

2 .List转换为数组

List<String> list = new ArrayList<String>();
list.add("str1");
list.add("str2");
int size = list.size();
String[] arr = list.toArray(new String[size]);
